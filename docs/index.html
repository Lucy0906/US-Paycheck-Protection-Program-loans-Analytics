US Paycheck Protection Program Loans Analytics

Analyzing the impact and utilization of PPP loans using Python and data visualization tools.

Project Overview

This project leverages data analytics to provide insights into the distribution, utilization, and impact of Paycheck Protection Program (PPP) loans during the COVID-19 pandemic. By analyzing PPP loan data, this project highlights trends, identifies key beneficiaries, and uncovers disparities in the program's rollout.

Key Features

  • Data Cleaning & Preparation: Preprocessed raw PPP loan data to ensure consistency and accuracy.
  • Geographic Analysis: Mapped loan distribution across states and industries using Python libraries.
  • Visualization: Created dynamic dashboards and static visualizations to communicate insights.
  • Industry Insights: Identified trends and disparities across different business sectors.

Tools & Technologies

  • Python (Pandas, NumPy, Matplotlib, Seaborn)
  • Tableau
  • Power BI
  • GeoPandas
  • SQL
  • Public PPP Loan Dataset

Use Cases

This project is designed to support policymakers, business owners, and analysts in understanding the effectiveness of PPP loans. It can help uncover disparities, optimize future programs, and provide transparency in fund allocation.
